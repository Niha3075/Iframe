1. What is HTML?
    2.
    • HTML stands for HyperText Markup Language.
    • It is the standard language used to create and design web pages.
      1. HyperText:
        Refers to the way you navigate the web—using links (called hyperlinks).
      2. Markup Language:
        It uses tags to define elements in a document, like headings, paragraphs, images, and links.
  2. What is Tag?
    3.
    • A tag is a piece of code in HTML used to define the structure and content of a web page.
    • Type of Tag
      1. Paired Tags (with opening and closing):
        • Example:h1 tag, p tag
        • Must be closed with tagname
      2. Self-closing Tags (no closing tag needed):
        • Example:h1 tag, p tag
        • Must be closed with tagname
  3. What is Element?
    4.

  8. What is XML and HTML
    9.
    1. HTML
      2.
      • HTML (HyperText Markup Language) is a markup language used to create and display content on web pages.
      • It focuses on the structure and presentation of information, using predefined tags such as p tag for paragraphs, h1 tag for headings, and < img > for images.
      • HTML tells the browser how the content should appear but is not designed for storing or transporting data. It is not case-sensitive and is somewhat forgiving of minor syntax errors.
    2. XTML
      3.
      • XML (Extensible Markup Language), on the other hand, is designed to store and transport data.
      • HTML, XML does not have predefined tags — you create your own tags to describe the data, such as < book >, < title >, and < author >.
      • XML focuses on the meaning and structure of the data rather than its presentation. It is case-sensitive, and its syntax rules are strict, requiring every tag to be properly opened, closed, and nested.
